## Run local

Clone the project

```bash
  git clone https://github.com/yassinehaimouch/Anime-Quotes.git
```

Enter the project directory 

```bash
  cd Anime-Quotes
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
